# Soma de Números Positivos em Java

Este projeto tem como objetivo ler números inteiros positivos informados pelo usuário e calcular a soma total desses valores.

## 📌 Como funciona

O programa solicita ao usuário a entrada de números inteiros.  
A soma é realizada enquanto os números digitados forem positivos.

O processo é encerrado quando o usuário digita o valor **-1**, que funciona como condição de parada.

## 🎯 Objetivo

Praticar:
- Estrutura de repetição `while`
- Entrada de dados com `JOptionPane`
- Controle de fluxo com condição de parada
- Acumulação de valores

## 💻 Tecnologias utilizadas

- Java
